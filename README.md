# CyberHospital

CyberHospital es un sistema de gestión hospitalaria desarrollado en Java utilizando Maven. Este proyecto permite administrar doctores, pacientes, citas y administradores.

## Instalación y Configuración

Sigue estos pasos para instalar y configurar el proyecto CyberHospital:

1. **Clona el repositorio:**
   ```sh
   git clone https://github.com/Icuruya/CyberHospital.git
   cd CyberHospital

2. **Uso del programa**

Uso del Programa
Inicio de Sesión:
Al iniciar el programa, se te pedirá que ingreses el nombre de usuario y la contraseña de un administrador registrado.

Menú Principal:

Dar de alta doctores.
Dar de alta pacientes.
Crear una cita.
Relacionar una cita con un doctor y un paciente.
Agregar nuevo administrador.
Salir del sistema.

Agregar Nuevo Administrador:

Selecciona la opción para agregar un nuevo administrador y sigue las indicaciones para ingresar el nombre de usuario y la contraseña.

Gestión de Doctores, Pacientes y Citas:

Sigue las opciones del menú principal para dar de alta doctores y pacientes, y para crear y gestionar citas.

2.5 **Compilar proyecyo utilizando Maven**
-   mvn clean install
-   Ejecutas programa
-   mvn exec:java -Dexec.mainClass="com.cyberhospital.CyberHospital"
